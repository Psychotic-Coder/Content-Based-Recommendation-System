# Content-Based-Recommendation-System
PROJECT: LAPTOP RECOMMENDER SYSTEM <br />

// DEVELOPED ON UBUNTU <br />
// WEB-APPLICATION FOR LAPTOP RECOMMENDATION BASED ON USER BROWSING HISTORY <br />

### TECHNOLOGIES USED:
	- PYTHON
	- ANGULAR CLI
<br />
PYTHON IS USED FOR BACKEND, BASED ON FLASK FRAMEWORK.<br />
ANGULAR IS USED FOR FRONTEND.<br /><br />

// SETUP<br />
### PRE-REQUISITES:
	- python version >=3.6.8
	- pip version >=9.0.1
	- npm version >=6.11.3
	- ng version >=8.3.14
	- stable internet

### RUN BACKEND SERVER:
	- cd into direcctory
	- pip3 install -r requirements.txt
	- python3 server.py

### RUN FRONTEND SERVER:
	- cd into directory
	- npm install
	- ng serve
